# Loki Setup & Log Viewing

1. How I started Loki:
I ran Loki locally via Docker using this command:
docker run -d --name loki -p 3100:3100 grafana/loki:3.0.0

To allow Docker to forward logs to Loki, I installed the driver:
docker plugin install grafana/loki-docker-driver:latest --alias loki --grant-all-permissions

2. The command I used to view logs:
To view the logs generated by the Nomad container ("Hello, DevOps!"), I used curl to query the Loki API for the specific job label:
curl -G -s "http://localhost:3100/loki/api/v1/query" --data-urlencode 'query={job="nomad-hello-app"}'
